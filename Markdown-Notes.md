## Markdown Notes
Markdown is a light weight markup language for authoring readable, functional code on the fly
- "`#`"  level one heading
- "`###`"  level three heading


### Multiple Bullets, One List
- **this text is _important_**
- ***"`***`"  is bold and italicized***
- "`~~`" ~~is deleted~~
- "`* *`"  *is italic* 
- **"`**`"  is bold**
